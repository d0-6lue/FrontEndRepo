배열

1. 배열 선언 및 초기화

        자바스크립트에서 배열은 자료형을 지정하지 않기때문에 한 배열 안에 모든 자료형의 데이터 보관이 가능하다.
        배열 선언 시 크기를 지정하지 않고 선언할 수도 있고 크기를 정하고 선언할 수도 있다.

2. 배열의 메소드

        배열도 하나의 객체이므로, 메소드들을 가짐.

        indexOf() : 특정 요소의 인덱스 반환
        concat() : 배열을 합칠 때 사용
        join() : 특정 값을 요소 사이에 넣어서 '문자열'을 만들고 반환
        reverse() : 원본 배열을 뒤집음
        sort() : 문자열 기준 오름차순으로 정렬
        sort(정렬방법(함수)) : 
        push() : 배열의 맨 뒤에 요소 추가
        pop() : 배열의 맨 뒤 요소를 반환, 원본에서 제거
        shift() : 배열의 맨 앞 요소를 반환, 원본에서 제거
        unshift() : 요소를 왼쪽으로 한칸씩 이동 ( 배열의 앞에 요소 추가 )
        slice(a [, b]) : a 이상 b 미만 인덱스를 가진 요소가 반환
        splice(a [, b]) : a 부터 b개의 요소를 반환, 원본에서 제거
        splice(a , 0, ... ) : a 인덱스 요소 뒤에 ... 추가
