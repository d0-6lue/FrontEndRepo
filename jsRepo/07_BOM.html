BOM ( Browser Object Model )

window 객체

        window 객체는 자바스크립트의 최상위 객체이며, 브라우저 내의 모든 요소들과 객체를 포함한다.
        BOM ( Browser Object Model ) : location 객체, navigator 객체, history 객체, screen 객체
        DOM ( Documnet Object Model ) : document 객체

1) window.open()

2) 타이머

- window.setTimeout()

        일정 시간 이후 동작

- window.setInterval()

        일정 시간 간격 동작

BOM 객체

1) location

        브라우저의 주시표시줄 관련 객체

        href        : URL 이동
        reload()    : 새로고침
        assign()    : 페이지 이동
        replace()   : 페이지 이동(기존 페이지를 대체, 히스토리에 기존 페이지가 남지 않음)

2) history

        브라우저 방문 기록 관련 객체

3) navigator

        브라우저 정보를 담고있는 객체

4) screen

        운영체제의 화면(모니터) 정보를 가지고있는 객체
