객체

객체 선언

        new Object() 또는 {} 를 이용해서 선언
        key : value 형태로 데이터를 담음(속성명 : 속성값)
            
            -속성명에 공백이나 특수문자 등이 포함 된 경우, 따옴표로 묶어서 표기
            -접근할 때 대괄호 이용.

객체 메소드

        객체의 속성(property) 중 자료형이 함수인 속성.

객체와 반복문

        for( in ) : 객체의 모든 key에 접근
        객체의 모든 키에 접근

객체의 속성 추가 및 제거

        객체를 생성한 이후,
        동적으로 속성 추가 및 삭제 가능

객체 배열

        객체를 배열 만들기

생성자 함수

        new 키워드를 사용해 객체를 생성할 수 있는 함수

캡슐화

        생성자 함수에 속성을 정의할 때,
        this. 키워드를 사용하지 않고, 지역변수로 선언
        지역변수에 접근 가능한 메소드 작성(getter/setter)
